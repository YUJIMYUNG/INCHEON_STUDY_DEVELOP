## 이론

---

```css
누구를 { 무엇을 : 어떻게 }

h1. { color : red }
 .ltaly { color : pink }
 .swiss { color : green }
```

- 누구를  : **선택자**
- 무엇을 : **속성**
- 어떻게 : **값**
- 별명, 즉 클래스를 타켓팅 할 때는 .을 찍어서 클래스를 식별한다.
