## 형변환

<aside>
💡

정수와 실수는 컴퓨터 내부에서 표현하는 방식이 전혀 다르다. 따라서 정수와 실수를 더한다고 할 때 그대로 연산을 수행할 수 없고 하나의 자료형으로 통일한 후 연산을 해야한다.

이때 **형 변환(type conversion)**이 이루어진다.

</aside>

1. 묵시적 형변환

- 자동 형변환(묵시적 형변환) : 작은 자료형 → 큰 자료형, 정수형 → 실수형
- 연산 중 자동 형변환 : 연산은 같은 자료형만 가능

1. 명시적 형변환(강제 형변환)

- 데이터의 유실이 발생할 가능성이 있는 경우 - 자동 형변환 X
- 명시적으로 형변환 의사 표현

```java
public class Ex04 {
    public static void main(String[] args) {
        int num1 = 100000; // 4byte
        byte num2 = (byte)num1; // 명시적 형변환 - 강제 형변환 (데이터 유실 발생 가능)
        System.out.println(num2); //-96출력됨 - 데이터 변질

    }
}
```
