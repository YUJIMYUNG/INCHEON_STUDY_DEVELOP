# 자바스크립트

- 브라우저를 조작하는 프로그래밍 언어

# 변수

- 데이터를 저장하는 공간
- const / let / var(X)
- 컴퓨터는 기억과 명령을 따로하기 때문에 변수를 사용한다.

# 데이터 타입

- 기본 타입 : String, Number, Boolean, Null, Undefined
- 참조 타입 : Array [], Object {key:value}

# 연산자

- 산술연산자, 논리연산자, 비교연산자, 대입연산자, 삼항연산자 ...
- 논리연산자에서 단축평가(short-circuit)을 사용해서 코드를 짧게 쓸 수 있다.

# 조건문

- 어떠한 조건에 의하여 실행되는 구문
- if ~else
